# نظام ناجز

نظام إدارة الطلبات يسمح للمستخدمين بتسجيل الدخول، إنشاء حسابات، رفع طلبات، ومتابعتها. كما يتيح للمدير إدارة الطلبات.

## الميزات:
- تسجيل دخول وإنشاء حساب.
- رفع ملفات وطلبات.
- حساب مدير لإدارة الطلبات.
